# GraphRAG UI Verification Prompts (測試問題集)

This document contains 100 English and 100 Mandarin (Traditional) prompts designed perfectly to stress-test your Enterprise GraphRAG Frontend UI. These questions are tuned to trigger both Vector Search (BM25 + Semantic) and Multi-Hop Cypher Context (Reviews, Features, Scenarios).
